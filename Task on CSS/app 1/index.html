What is CSS? CSS (Cascading Style Sheets) is a stylesheet language used to describe the look and formatting of a document written in HTML (or sometimes XML). It controls the visual presentation of web pages — how elements appear in terms of layout, colors, fonts, spacing, and responsiveness.

Core purpose of CSS

While HTML provides the structure of a webpage (e.g., paragraphs, headings, images), CSS is responsible for the styling — such as making text red, aligning content, or making a layout responsive for different devices.

How CSS Works

CSS works by applying rules to HTML elements. A rule consists of a selector (which element to target) and a declaration block (the style to apply).
CSS (Cascading Style Sheets) is a language used to style and format HTML elements on a webpage. It works by selecting elements and applying rules that define their appearance, such as colors, fonts, spacing, and layout. CSS can be written inline, inside a style tag, or in an external file. The browser reads both the HTML structure and CSS styles, then renders the content with the applied design. This separation of content and style makes web development cleaner and more efficient.

Types of CSS

  1. Inline CSS – added directly inside HTML elements using the style attribute.
    Example: p style="color:red;"
  2. Internal CSS – placed inside a style tag within the style of the HTML file.
    Example: style
    h1 { color: green; }
    /style
  3. External CSS – written in a separate .css file and linked to the HTML.
    Example: link rel="stylesheet" href="style.css"

Why Use CSS?

  • Separates content (HTML) from presentation (CSS).
  • Makes websites more maintainable and scalable.
  • Supports responsive and accessible design.
  • Enables rich user interfaces with minimal code.
