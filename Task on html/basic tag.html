Basic Tags

HTML (HyperText Markup Language) uses a variety of basic tags to structure and display content on web pages.

The !DOCTYPE tag defines the HTML version being used.

The html tag wraps the entire document, while the head contains metadata such as the title, which sets the page title shown in the browser tab.

The body tag holds all visible content, including headings (h1 to h6), paragraphs (p), and links (a).

Lists can be created using ul for unordered and ol for ordered lists, with each item placed inside li.

Images are added using the img tag, and line breaks with br.

Containers like div (block-level) and span (inline) help group elements for layout or styling.These basic tags form the foundation of any HTML document, enabling web developers to create structured and meaningful content.
