Page format and Images

Page format

Formatting tags such as strong (bold) and em (italic) are used to emphasize text.

Images

In HTML, images are added to a webpage using the img tag. This tag is used to embed visual content such as photos, graphics, or icons, and it does not require a closing tag. The most important attribute of the img tag is src (source), which specifies the path or URL of the image file. Another essential attribute is alt (alternative text), which provides a text description of the image in case it cannot be displayed or for screen readers used by visually impaired users.

For example: img src="logo.png" alt="Company Logo" will display the image named "logo.png" and show "Company Logo" as alt text if the image fails to load. Additional attributes like `width` and `height` can be used to control the image's size. Using images effectively enhances the visual appeal and user experience of a website.
